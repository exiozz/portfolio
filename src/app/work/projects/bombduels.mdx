## Overview

Bomb Duels is a fast-paced competitive experience developed on the Roblox platform. My goal was to create fast-paced and balanced gameplay while offering an immersive user interface (UI). This project allowed me to combine my skills in software engineering (Luau) and interface design to provide players with a seamless experience, from the lobby to the heart of the arena.
## Key Features

- **Dynamic Combat System**: Developed a combat system based on projectile physics, providing optimal precision and responsiveness.
- **Immersive UI/UX**: Designed an intuitive game interface (HUD, selection menus) that facilitates immersion and quick understanding of mechanics by the player.
- **Performance-First Architecture**: Optimized server-side and client-side code to ensure a stable framerate, even with multiple simultaneous explosion effects.
- **Scalable Matchmaking**: Implemented a dynamic game management system to handle player entry and exit in real-time without interruption.

## Technologies Used

- **Roblox Studio & Luau**: For business logic, game physics, and 3D world management.
- **Figma**: For UI design and game flow prototyping.
- **DataStore Service**: For player data persistence, statistics, and unlocked items.
- **Adobe Photoshop/Illustrator**: For creating custom visual assets.

## Challenges and Learnings

The major challenge was performance optimization under high intensity. Managing dozens of bombs and visual effects without dropping the frame rate was critical. This project taught me how to handle synchronization between the server and clients and how to optimize memory so that the game remains fluid on lower-end mobile devices.

## Outcome

Bomb Duels now offers a solid competitive experience with an active player base. By adopting a rigorous "Game Design Engineering" approach, I managed to transform a simple concept into a high-performance, visually cohesive, and technically robust game. This project demonstrates my ability to manage the full product lifecycle, from visual design to technical deployment.
