## Overview

Stradaz Café & Bakery required a digital presence that matched the quality of its products. My goal was to create a web ecosystem that wasn't just a static showcase, but a true extension of the in-store experience. I designed and developed a user-centric platform that simplifies menu discovery and highlights the brand's visual identity through a fluid and modern interface.

## Key Features

- **Brand-Centric Design**: Created a digital visual identity consistent with the bakery’s physical environment, using carefully selected typography and color palettes to reinforce branding.
- **Performance & SEO-Optimized**: Developed with Next.js to ensure lightning-fast load times, which is essential for local customers searching via mobile.
- **Scalable Content Management**: Implemented an architecture that allows for easy management of the menu, photo gallery, and announcements without requiring constant technical maintenance.
- **Seamless User Journey**: Navigation architecture designed to minimize friction between online discovery and the physical visit.

## Technologies Used

- **Next.js & React**: Used to ensure fast execution and optimal SEO performance on search engines.
- **Tailwind CSS**: Used for a maintainable and responsive design system across all screen sizes.
- **Figma**: Used for the entire UI/UX design phase and interactive prototyping.
- **TypeScript**: Used to secure the codebase and allow for scalable, error-free development during future updates.

## Challenges and Learnings

The main challenge was balancing high visual fidelity (high-resolution, visually appealing images) with exemplary load speeds. I had to implement asset optimization strategies and intelligent rendering to maintain this "premium" aesthetic without penalizing mobile users. This project allowed me to perfect my approach to web performance optimization.

## Outcome

The Stradaz platform has become an essential communication tool for the business. It now offers an intuitive and professional user experience that reflects artisanal craftsmanship. This project demonstrates my ability to take ownership of a project from start to finish, from UI design to the deployment of a high-performance, robust product.
